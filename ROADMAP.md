# Roadmap (0–12 months)

0–3 months
- Recruit initial contributors and department leads.
- Publish first 1–3 research papers (monthly cadence).
- Release initial datasets and example notebooks.
- Run workshops and public seminars.

3–6 months
- Establish city-level partnerships and host larger events.
- Expand compute resources via grants/college allocations.
- Publish 3 more papers; iterate on model releases.

6–12 months
- Launch flagship LLM/CVM projects (open-source).
- Formalize Control Committee and secure external collaborations.
- Participate in national hackathons and conferences.

KPIs
- Publications: >= 6 papers in 12 months.
- Datasets: >= 5 curated public datasets.
- Models: >= 3 released model checkpoints with evaluation.
- Community: regular contributors, external collaborators, events hosted.
